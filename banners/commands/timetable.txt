timetable [rozvrh]

Protože jsi hloupá piča tady máš vysvětlení.
Tento příkaz vypíše aktuální rozvrh. Vypíše i čísla hodin
a názvy jednotlivých dní. V případě, že chceš zobrazit
stálý rozvrh nebo rozvrh z minulého či následujícího
týdne, využij správný option.

Options:
-s (stable)   -> stálý
-p (previous) -> minulý týden
-n (next)     -> další týden
-m (minimal)  -> minimální výpis
  (nevypíše čísla hodin a názvy dnů)
-r (rooms)    -> vypíše názvy tříd namísto předmětů
-t (teachers) -> vypíše názvy učitelů namísto předmětů
-d (dates)    -> vypíše čísla dnů

Použití:
-> timetable
-> timetable -n
-> timetable -pm
-> timetable -d
-> rozvrh
